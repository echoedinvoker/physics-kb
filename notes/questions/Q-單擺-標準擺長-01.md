# Q-單擺-標準擺長-01

## 題目

在相同時間內以長 $l_1$ 的單擺計時較標準擺快 t 秒，而長改為 $l_2$ 時較標準擺慢 t 秒，則標準擺之擺長為何？

## 答案

$$L = \frac{4l_1 l_2}{(\sqrt{l_1} + \sqrt{l_2})^2}$$

## 解析

設標準擺長為 L，在相同時間 τ 內：
- 快 t 秒：$\frac{\tau}{T_1} - \frac{\tau}{T_0} = t$（擺 1 多擺了 t 秒的次數）
- 慢 t 秒：$\frac{\tau}{T_0} - \frac{\tau}{T_2} = t$

由兩式相加：$\frac{1}{T_1} - \frac{1}{T_2} = \frac{2}{T_0}$

代入 $T \propto \sqrt{L}$：$\frac{1}{\sqrt{l_1}} + \frac{1}{\sqrt{l_2}} = \frac{2}{\sqrt{L}}$

$$\sqrt{L} = \frac{2\sqrt{l_1 l_2}}{\sqrt{l_1} + \sqrt{l_2}}$$

$$L = \frac{4l_1 l_2}{(\sqrt{l_1} + \sqrt{l_2})^2}$$
