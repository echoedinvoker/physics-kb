# Q-因次-物理量因次-01

## 題目

假定有某個物理量 x 可表為 x = ky + z，則：

(A) x 與 k 的單位須相同
(B) x 與 y 的單位須相同
(C) x 與 z 的單位須相同
(D) x 與 ky 的數量須相同
(E) x 與 z 的數量須相同

## 答案

C

## 解析

- **(C)** — 正確，加法運算中，x、ky、z 三者的因次（單位）必須相同。所以 x 與 z 的單位須相同

根據[[因次分析]]：等式兩邊及加減法各項的因次必須一致。

## 測試概念

- [[因次分析]] — 因次一致性原則：等式中加減項的因次必須相同
