# Q-因次-單擺因次分析-01

## 題目

小角度擺動的單擺，具有等時性，已知其週期 T 與擺長 L 以及重力加速度 g 有關，若其關係可表為 T = CL^a g^b，C 為無單位的常數。則：(1) 式中的 a 值為何？(2) 式中的 b 值為何？

## 答案

a = 1/2, b = -1/2

## 解析

由[[因次分析]]：
- T 的因次：$[T]$
- L 的因次：$[L]$
- g 的因次：$[LT^{-2}]$

$[T] = [L]^a [LT^{-2}]^b = [L^{a+b}][T^{-2b}]$

比較因次：
- L：$a + b = 0$
- T：$-2b = 1 \Rightarrow b = -\frac{1}{2}$
- 代入：$a = \frac{1}{2}$

所以 $T = C\sqrt{L/g}$，與[[單擺週期公式]] $T = 2\pi\sqrt{L/g}$ 一致。
