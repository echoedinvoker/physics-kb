# Q-長度-鉛筆基準面積-01

## 題目

某生以鉛筆的長度為基準，測量桌子的長與寬，其讀數分別為 a 與 b。若鉛筆長度以標準尺測量時的讀數為 c，則以標準尺的單位來表示桌子面積時，其結果應為______。

## 答案

$abc^2$

## 解析

根據[[測量與誤差]]中的單位換算：

- 鉛筆長度 = c（標準尺單位）
- 桌子長 = a 支鉛筆 = $a \times c$（標準尺單位）
- 桌子寬 = b 支鉛筆 = $b \times c$（標準尺單位）

面積 = 長 $\times$ 寬 = $(ac)(bc) = abc^2$

## 測試概念

- [[測量與誤差]] — 非標準單位的換算：自訂基準量測後轉換為標準單位
