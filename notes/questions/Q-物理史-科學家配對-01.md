# Q-物理史-科學家配對-01

## 題目

下列物理理論或現象的發現者是誰？以右邊物理學家，選出正確的答案填入。

1.( ) 提出「慣性」。
2.( ) 提出「日心說」。
3.( ) 提出「萬有引力定律」。
4.( ) 提出「相對論」。
5.( ) 提出「量子論」。
6.( ) 第一個發現 X 光的科學家。
7.( ) 提出「行星三大運動定律」。
8.( ) 提出光的「波動說」。
9.( ) 發現電子的科學家。
10.( ) 提出「電磁感應定律」。

選項：(A)托勒密 (B)哥白尼 (C)莫谷 (D)克卜勒 (E)牛頓 (F)伽利略 (G)愛因斯坦 (H)卜朗克 (I)湯木生 (J)海更士 (K)侖琴 (L)富蘭克林 (M)菲左 (N)焦耳 (O)居里夫婦 (P)庫侖 (Q)安培 (R)法拉第 (S)楊格 (T)海森堡

## 答案

1.F 2.B 3.E 4.G 5.H 6.K 7.D 8.J 9.I 10.R

## 解析

- 1. **慣性**：[[伽利略]](F) 透過斜面實驗發現物體保持原運動狀態的性質
- 2. **日心說**：[[哥白尼]](B) 提出太陽為宇宙中心
- 3. **萬有引力定律**：[[牛頓]](E) 1687 年提出
- 4. **相對論**：[[愛因斯坦]](G) 1905 年狹義、1915 年廣義
- 5. **量子論**：[[卜朗克]](H) 1900 年提出能量量子化
- 6. **X 光**：[[侖琴]](K) 1895 年發現
- 7. **行星三大定律**：[[克卜勒]](D) 根據[[第谷]]數據歸納
- 8. **波動說**：[[海更士]](J) 提出光的波動說
- 9. **電子**：[[湯木生]](I) 1897 年發現
- 10. **電磁感應定律**：[[法拉第]](R) 1831 年發現

## 測試概念

此題測試對物理學史上重要科學家及其核心貢獻的配對能力。

## 解題策略

> 此段為 LLM 增強內容。

建立「科學家—核心貢獻」的對照表是準備此類題型的最有效方式。特別注意容易混淆的：
- 伽利略 vs 牛頓：伽利略發現慣性，牛頓將其納入第一定律
- 哥白尼 vs 克卜勒：哥白尼提出日心說，克卜勒修正為橢圓軌道
