# Q-有效數字-有效位數-01

## 題目

下列各量中，各有多少位有效數字？

(A) 3.150cm (B) 0.0315cm (C) 6.025cm (D) 36km (E) 3.34×10⁶ (F) 36.00

## 答案

(A) 4 (B) 3 (C) 4 (D) 2 (E) 3 (F) 4

## 解析

根據[[有效數字]]的規則：
- **(A) 3.150** → 4 位（小數點最後的 0 算有效數字）
- **(B) 0.0315** → 3 位（純小數前的 0 不算）
- **(C) 6.025** → 4 位（兩個非零數字間的 0 算有效）
- **(D) 36** → 2 位
- **(E) 3.34×10⁶** → 3 位
- **(F) 36.00** → 4 位（小數點後的 0 算有效）

## 測試概念

- [[有效數字]] — 有效數字位數判斷規則

## 解題策略

> 此段為 LLM 增強內容。

三個易錯點：(1) 小數點最後的 0 是有效的（如 3.150 有 4 位）；(2) 純小數前導 0 不算（如 0.0315 只有 3 位）；(3) 整數末尾的 0 不算（如 1000 只有 1 位，但 1000. 有 4 位）。
