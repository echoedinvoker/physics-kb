# Q-單擺-擺動加快-01

## 題目

一作小角度擺動的單擺，下列什麼原因會使其擺動加快？

(A) 溫度下降，擺長改變了
(B) 搭應地球自轉，從低緯度的地方搬到高緯度的地方使用
(C) 從高山上搬到平地使用
(D) 放在電梯中，電梯向上加速運動
(E) 從地球表面移到月球表面使用

## 答案

ABCD

## 解析

- **(A)** — 正確，溫度下降 → 擺長縮短 → 週期縮短 → 加快
- **(B)** — 正確，高緯度 g 較大 → 週期縮短 → 加快
- **(C)** — 正確，平地 g 較高山大 → 週期縮短 → 加快
- **(D)** — 正確，電梯向上加速 → 等效 g 增大 → 週期縮短 → 加快
- **(E)** — 錯誤，月球 g 較地球小 → 週期增長 → 變慢
