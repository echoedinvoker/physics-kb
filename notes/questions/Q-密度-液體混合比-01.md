# Q-密度-液體混合比-01

## 題目

有 a、b 兩種液體，其密度分別為 $d$、$2d$。今將兩液體混合時，混合液體的密度為 $\frac{6}{5}d$。假定混合前後的體積總和不變，則 a、b 兩液體原來的體積整數比為？

## 答案

$4:1$

## 解析

- 設 a 液體體積為 $V_a$，b 液體體積為 $V_b$
- 混合[[密度]]（體積守恆）：
  $\rho_{\text{mix}} = \frac{d \cdot V_a + 2d \cdot V_b}{V_a + V_b} = \frac{6}{5}d$
- 兩邊除以 $d$：
  $\frac{V_a + 2V_b}{V_a + V_b} = \frac{6}{5}$
- 交叉相乘：$5(V_a + 2V_b) = 6(V_a + V_b)$
- $5V_a + 10V_b = 6V_a + 6V_b$
- $4V_b = V_a$
- $V_a : V_b = 4 : 1$

## 測試概念

- [[密度]] — 由混合密度反推液體體積比
