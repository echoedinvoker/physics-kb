# Q-三角-河寬-01

## 題目
如附圖所示，甲、乙兩人利用[[三角測量]]法，欲測河川的寬度。已知甲、乙兩人以彼此間的距離為基線，而與對面岸上某定點的觀測角度分別為 37° 與 53°，若甲、乙兩人相距 200 公尺，則河寬為若干公尺？
(A)64 (B)82 (C)96 (D)112 (E)48

## 答案
(C)96 公尺

## 解析
設河寬為 $h$，甲到對岸定點正對面的水平距離為 $x$，則乙到該點正對面的水平距離為 $200 - x$。

由甲的觀測角 37°：
$$\tan 37° = \frac{h}{x} \implies x = \frac{h}{\tan 37°} = \frac{h}{0.75}$$

由乙的觀測角 53°：
$$\tan 53° = \frac{h}{200 - x} \implies 200 - x = \frac{h}{\tan 53°} = \frac{h}{1.333}$$

兩式相加：
$$200 = \frac{h}{0.75} + \frac{h}{1.333} = h\left(\frac{4}{3} + \frac{3}{4}\right) = h \times \frac{25}{12}$$

$$h = 200 \times \frac{12}{25} = 96 \text{ m}$$

## 測試概念
- [[測量與誤差]] — 三角測量法測定不可直接到達的距離
