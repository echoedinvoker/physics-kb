# Q-測量-誤差計算-01

## 題目

甲在緯度 45° 的海面上做重力加速度 g 之測量，已知最大可能值為 980 cm/s²，甲測得 978 cm/s²。試求甲的絕對誤差、相對誤差、準確度。

## 答案

- 絕對誤差 = 978 - 980 = **-2 cm/s²**
- 相對誤差 = -2/980 = **-0.204%**
- 準確度 = 1 - |−2/980| = **99.8%**

## 解析

根據[[測量與誤差]]的定義：
- **絕對誤差** $\Delta A = A - A_0 = 978 - 980 = -2$ cm/s²
- **相對誤差** = $\frac{\Delta A}{A_0} = \frac{-2}{980} = -0.00204 = -0.204\%$
- **準確度** = $1 - \left|\frac{\Delta A}{A_0}\right| = 1 - 0.00204 = 99.8\%$
