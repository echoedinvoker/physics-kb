# Q-密度-混合密度判斷-01

## 題目

密度為 $\rho_1$ 與 $\rho_2$ 的兩液體，且 $\rho_1 > \rho_2$，則下列何者正確？

(A) 取等質量混合後（假定體積守恆），混合密度為 $\frac{\rho_1 + \rho_2}{2}$
(B) 取等體積混合後（假定體積守恆），混合密度為 $\frac{2\rho_1 \rho_2}{\rho_1 + \rho_2}$
(C) 兩液體任意混合，密度值可以為任意數
(D) 等體積混合的密度比等質量混合的密度大

## 答案

D

## 解析

**等質量混合**（各取質量 $m$）：
- 總質量 = $2m$，總體積 = $\frac{m}{\rho_1} + \frac{m}{\rho_2}$
- $\rho_{\text{等質}} = \frac{2m}{\frac{m}{\rho_1} + \frac{m}{\rho_2}} = \frac{2\rho_1 \rho_2}{\rho_1 + \rho_2}$（**調和平均**）

**等體積混合**（各取體積 $V$）：
- 總質量 = $\rho_1 V + \rho_2 V$，總體積 = $2V$
- $\rho_{\text{等體}} = \frac{(\rho_1 + \rho_2)V}{2V} = \frac{\rho_1 + \rho_2}{2}$（**算術平均**）

- **(A)** 這是等體積混合的結果，不是等質量混合 ✗
- **(B)** 這是等質量混合的結果，不是等體積混合 ✗
- **(C)** 混合[[密度]]必介於 $\rho_1$ 與 $\rho_2$ 之間 ✗
- **(D)** 算術平均 ≥ 調和平均（AM-HM 不等式），故 $\rho_{\text{等體}} \geq \rho_{\text{等質}}$，等號成立於 $\rho_1 = \rho_2$ ✓

## 測試概念

- [[密度]] — 等質量混合為調和平均，等體積混合為算術平均，算術平均 ≥ 調和平均
