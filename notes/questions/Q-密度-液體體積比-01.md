# Q-密度-液體體積比-01

## 題目

有 a、b 兩種液體，其密度分別為 $\rho_1$、$\rho_2$。今將兩液體混合時，混合液體的密度為 $\frac{\rho_1 + 2\rho_2}{3}$。假定混合前後的體積總和不變，則 a、b 兩液體原來的體積比為何？

## 答案

$V_a : V_b = 1 : 2$

## 解析

- 設 a 液體體積為 $V_a$，b 液體體積為 $V_b$
- 混合[[密度]]（體積守恆）：
  $\rho_{\text{mix}} = \frac{\rho_1 V_a + \rho_2 V_b}{V_a + V_b} = \frac{\rho_1 + 2\rho_2}{3}$
- 交叉相乘：$3(\rho_1 V_a + \rho_2 V_b) = (\rho_1 + 2\rho_2)(V_a + V_b)$
- 展開左邊：$3\rho_1 V_a + 3\rho_2 V_b$
- 展開右邊：$\rho_1 V_a + \rho_1 V_b + 2\rho_2 V_a + 2\rho_2 V_b$
- 移項整理：$(3\rho_1 - \rho_1 - 2\rho_2)V_a = (\rho_1 + 2\rho_2 - 3\rho_2)V_b$
- $(2\rho_1 - 2\rho_2)V_a = (\rho_1 - \rho_2)V_b$
- $2(\rho_1 - \rho_2)V_a = (\rho_1 - \rho_2)V_b$
- 因 $\rho_1 \neq \rho_2$，消去 $(\rho_1 - \rho_2)$：$2V_a = V_b$
- 故 $V_a : V_b = 1 : 2$
- **驗算**：設 $V_a = 1, V_b = 2$，$\rho = \frac{\rho_1 \cdot 1 + \rho_2 \cdot 2}{1 + 2} = \frac{\rho_1 + 2\rho_2}{3}$ ✓

## 測試概念

- [[密度]] — 由混合密度公式反推液體體積比
