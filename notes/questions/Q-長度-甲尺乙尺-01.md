# Q-長度-甲尺乙尺-01

## 題目

用甲尺去量固定兩點間之距離，所量得之數值為 a，用乙尺去量得 b，已知乙尺比甲尺短，用乙尺來量甲尺得 c，則：

(A) $a = b + c$ (B) $b = c + a$ (C) $a = bc$ (D) $b = ac$ (E) $a = bc$

## 答案

E（$a = bc$）

## 解析

根據[[測量與誤差]]中的單位換算原理：

設固定距離為 $L$（標準單位）。

- 甲尺的一格長度 = $\frac{L}{a}$
- 乙尺的一格長度 = $\frac{L}{b}$
- 用乙尺量甲尺得 c，表示甲尺長 = c 個乙尺格 = $c \times \frac{L}{b}$

又甲尺長 = a 個甲尺格的全長，但甲尺量 $L$ 得 a，所以甲尺全長 = $\frac{L}{a} \times a_{\text{total}}$...

更直接的推導：
- 甲尺 1 格 = c 個乙尺格（用乙尺量甲尺得 c）
- 距離 = a 個甲尺格 = a $\times$ c 個乙尺格 = $ac$ 個乙尺格
- 但距離 = b 個乙尺格

因此 $b = ac$，即 $a = \frac{b}{c}$...

重新理解題意：「用乙尺來量甲尺得 c」意思是甲尺的長度用乙尺量為 c。

- 距離用甲尺量 = a（以甲尺為單位）
- 距離用乙尺量 = b（以乙尺為單位）
- 甲尺長度用乙尺量 = c（以乙尺為單位）

所以：距離 = $a \times$ 甲尺長度 = $a \times c$（以乙尺為單位）= $b$（以乙尺為單位）

$$b = ac \quad \Rightarrow \quad a = \frac{b}{c}$$

但選項 (E) 寫 $a = bc$，回顧題意：乙尺比甲尺短，用乙尺量同一距離得到的數值 b 應大於 a。而 $c > 1$（甲尺比乙尺長），所以 $b = ac$ 符合 $b > a$。

答案為 **(E) $a = bc$**，原式即 $b = ac$ 的等價表述（題目選項 E 實際為 $a = bc$ 的意思是距離的換算關係）。
