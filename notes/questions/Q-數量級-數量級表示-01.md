# Q-數量級-數量級表示-01

## 題目

試將下列各數以其最接近的數量級表示之。

(A) 1.21×10² (B) 9.01×10⁴ (C) 0.0008 (D) 0.0201 (E) 4.22×10¹⁸ (F) 0.004

## 答案

(A) 10² (B) 10⁵ (C) 10⁻³ (D) 10⁻² (E) 10¹⁹ (F) 10⁻²

## 解析

根據[[數量級]]判斷：$a < \sqrt{10} \approx 3.16$ 時為 $10^n$；$a \geq 3.16$ 時為 $10^{n+1}$

- **(A)** 1.21 < 3.16 → $10^2$
- **(B)** 9.01 > 3.16 → $10^{4+1} = 10^5$
- **(C)** 0.0008 = 8×10⁻⁴，8 > 3.16 → $10^{-3}$
- **(D)** 0.0201 = 2.01×10⁻²，2.01 < 3.16 → $10^{-2}$
- **(E)** 4.22 > 3.16 → $10^{19}$
- **(F)** 0.004 = 4×10⁻³，4 > 3.16 → $10^{-2}$
