# Q-單擺-甲乙二擺-01

## 題目

甲乙二擺，甲擺動 3 次的時間內乙恰擺動 4 次，若甲之擺長為 0.36m，則乙之擺長為：

(A) 0.64m (B) 0.48m (C) 0.27m (D) 0.20m

## 答案

D

## 解析

- 甲 3 次 = 乙 4 次 → 3T甲 = 4T乙 → T甲/T乙 = 4/3
- $T \propto \sqrt{L}$ → $\frac{T_甲}{T_乙} = \sqrt{\frac{L_甲}{L_乙}}$
- $\frac{4}{3} = \sqrt{\frac{0.36}{L_乙}}$
- $\frac{16}{9} = \frac{0.36}{L_乙}$ → $L_乙 = 0.36 \times \frac{9}{16} = 0.2025 ≈ \textbf{0.20m}$
