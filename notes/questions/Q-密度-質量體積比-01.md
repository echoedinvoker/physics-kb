# Q-密度-質量體積比-01

## 題目

質量相等的 A、B、C 三物體，其密度比為 $3:2:1$，則體積比為：

(A) $3:2:1$ (B) $1:2:3$ (C) $2:3:6$ (D) $6:3:2$

## 答案

C

## 解析

- 設質量均為 $m$，[[密度]]分別為 $3\rho$、$2\rho$、$\rho$
- $V = \frac{m}{\rho}$
- $V_A : V_B : V_C = \frac{m}{3\rho} : \frac{m}{2\rho} : \frac{m}{\rho}$
  $= \frac{1}{3} : \frac{1}{2} : 1$
- 通分（乘以 6）：$= 2 : 3 : 6$
- 故選 **(C)**

## 測試概念

- [[密度]] — 等質量下密度與體積成反比
