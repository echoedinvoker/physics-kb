# Ch1 物理學與人類生活 MOC

> 本章涵蓋物理學的歷史發展、物理學對人類生活的影響、以及基本的物理測量方法。

## 章節地圖

- [[Ch1-1 物理學簡史 MOC]] — 從古希臘到近代物理的發展歷程
- [[Ch1-2 物理學對人類生活的影響 MOC]] — 半導體、雷射、光纖、超導體、核能
- [[Ch1-3 物理學與測量 MOC]] — 測量方法、單位系統、誤差分析

## 學習重點

1. **物理學簡史**：認識重要科學家及其貢獻，理解物理學各分支的發展脈絡
2. **科技應用**：了解物理學如何轉化為改變人類生活的技術
3. **測量基礎**：掌握國際單位制、有效數字、誤差、因次分析等基本工具

## 各分支概覽

### 古典物理（五大分支）
- 力學：[[亞里斯多德]] → [[伽利略]] → [[牛頓]] → [[愛因斯坦]]
- 光學：[[司乃耳]] → [[牛頓]]/[[海更士]] → [[楊格]] → [[愛因斯坦]]
- 熱學：[[侖福特伯爵]] → [[卡諾]] → [[焦耳]] → [[克勞乃士]] → [[能士特]]
- 電磁學：[[吉爾伯特]] → [[庫侖]] → [[厄司特]] → [[法拉第]] → [[馬克士威]]
- 天文學：[[托勒密]] → [[哥白尼]] → [[第谷]] → [[克卜勒]] → [[牛頓]]

### 近代物理
- [[侖琴]] → [[卜朗克]] → [[愛因斯坦]] → [[波耳]] → [[海森堡]]/[[薛丁格]] → [[蓋耳曼]]
