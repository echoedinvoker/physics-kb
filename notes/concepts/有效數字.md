# 有效數字

> 凡是有意義的數字皆稱它有效數字。有效數字的位數反映測量的精確程度，運算結果的有效位數不能超過原始數據中最少的有效位數。

## 背景知識

理解此概念前需要先掌握：
- [[測量與誤差]] — 測量值的組成和意義

## 詳細說明

**有效數字的位數**：

A. 為有效數字：
- 非 0 的數字皆算一位有效數字（1234 四位，578 三位）
- 夾在兩個非 0 數字間的 0，要算為一位（1204 四位，508 三位）
- 小數點最後的 0，算一位有效位數（12.30 四位，1.800 四位）

B. 不為有效數字：
- 純小數前的 0，不算有效數字（0.1234 四位，0.058 二位）
- 整數後面的 0，不算有效數字（1000 一位，570 二位）

**運算法則**：
- 加減法：利用直式運算來判斷，原則上只能取一位估計值
- 乘除法：積（商）的有效位數，與乘數或被乘數中有效位數最少的相等
- 開方運算：結果的有效位數與原量相同
- 非度量性的數字（如倍數或無理數），其有效數字視為無限多位

## 常見錯誤

> 此段為 LLM 增強內容，教材原文未提及。

1. **錯誤認知**：計算結果直接取最多位數就好
   - 正確理解：結果的有效位數不能超過原始數據中最少的，否則就是虛假精確度
2. **易混淆概念**：整數後面的 0 算不算有效數字
   - 區別：1000 只有一位有效數字，但 1000.0 就有五位（有小數點）
