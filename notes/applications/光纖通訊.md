# 光纖通訊

> 光纖是由高純度石英（SiO2）拉製而成的纖維，利用全反射原理將光信號限制在纖維內部傳播，實現高速大容量的通訊。

## 物理原理

- 光纖利用光的「全反射」原理
- 當光的入射方向和中心軸之間的角度小於某特定角時，光會在纖芯和包層界面處發生全反射
- 光被限制在光纖內部傳播，不會洩漏
- 相關概念：[[司乃耳定律公式]]

## 結構與運作

- 光纖材質：由純度極高的石英（SiO2）熔融後抽絲製成通通
- 結構：直徑約為 125 微米，含內外兩層——纖芯和包層
- 纖芯折射率大於包層折射率，使光在其中發生全反射

> 圖示描述：光纖橫截面示意圖，中心為纖芯（折射率較高），外層為包層（折射率較低）。光線在纖芯中以鋸齒狀路徑前進，在纖芯-包層界面處發生全反射。

## 發展歷史

- 1950 年代末期：首先應用於製作腎臟診療病灶的內視鏡
- 1966 年：英國科學家首先研究以光纖來進行遠程通訊
- 1970 年：美國康寧玻璃公司製出合乎通訊要求的高品質光纖
- 1994 年：台灣已完成本島的長途光纖通訊系統，並鋪設了台灣與瀾瀨湖之間的海底光纜系統
- 2010 年目標：將實現電話、戶線路全面光纖化的願景

## 日常生活應用

- 長途電話通訊
- 網路寬頻
- 海底電纜
- 醫療內視鏡

## 關鍵特性

| 特性 | 說明 |
|------|------|
| 通訊容量大 | 可同時傳輸大量數據 |
| 有利於長距離通訊 | 信號衰減小 |
| 不受電磁的干擾 | 光信號不受電磁波影響 |
| 重量輕 | 比傳統銅纜輕很多 |
| 體積小 | 直徑僅 125 微米 |
