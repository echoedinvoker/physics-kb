# 核能發電

> 核能發電是利用鈾原子核的分裂（核分裂）反應，在控制下進行連鎖反應，將釋放出的核能轉變為電能的技術。

## 物理原理

- 核分裂：以慢中子撞擊鈾-235 原子核，使其分裂為較輕的原子核，同時釋放出中子和巨大的能量
- 連鎖反應：分裂產生的中子再去撞擊其他鈾原子核，形成連鎖反應
- 能量來源：反應前後的質量差（質量虧損）轉變為能量，遵循[[質能互換公式]] $E=mc^2$
- 核分裂反應式（鈾-235 的一種分裂方式）：$^1_0n + ^{235}_{92}U \to ^{141}_{56}Ba + ^{92}_{36}Kr + 3^1_0n$

## 結構與運作

核電廠的基本結構：
1. 核燃料（鈾-235）
2. 減速劑（使中子減速為慢中子）
3. 控制棒（吸收多餘中子，控制反應速率）
4. 冷卻劑（帶走熱能）
5. 蒸氣渦輪發電機（將熱能轉為電能）

由[[費米]]建造的第一座原子核反應爐證明可在人為控制下進行鈾原子核分裂的連鎖反應。

## 發展歷史

- [[哈恩]]：發現鈾的核分裂現象
- [[費米]]：建造第一座可控核反應爐
- [[貝特]]：指出太陽能量來自核融合反應
- 1945 年：引爆第一顆原子彈
- 1951 年：試爆第一顆氫彈

## 日常生活應用

- 核能發電廠（全球約 10% 的電力來自核能）
- 核子醫學
- 太空探測器的核電池

## 關鍵特性

| 特性 | 說明 |
|------|------|
| 能量密度極高 | 1 kg 鈾分裂的能量約為 1 kg 煤的 200 萬倍 |
| 不排放溫室氣體 | 運轉時不產生 CO2 |
| 放射性廢料 | 反應生成物帶有很強的輻射性 |
| 質量虧損 | 反應前後的質量差轉化為能量 |
